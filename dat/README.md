- `deaths_1x1`: HMD Death counts by region, sex, year, and age via https://www.mortality.org/Data/ZippedDataFiles
- `exposures_1x1`: HMD Population exposures by region, sex, year, and age via https://www.mortality.org/Data/ZippedDataFiles
- `population`: HMD January 1st population counts by region, sex, year, and age via https://www.mortality.org/Data/ZippedDataFiles
- `11-population.rds`: WPP22 Fertility rates, constant mortality rate projections, and January 1st and midyear population counts by region, sex, year, and age via https://population.un.org/wpp
- `12-deaths.rds`: STMF weekly death counts by region, sex, year, and age via https://www.mortality.org/File/GetDocument/Public/STMF/Inputs/STMFinput.zip
- `13-netmigration.rds`: Eurostat EUROPOP2019 projection assumed netmigration counts by region, sex, year, and age via https://ec.europa.eu/eurostat/databrowser/product/view/PROJ_19NANMIG?lang=en
- `14-lifeexpectancy.rds`: Life expectancy by region, sex, year, and age via https://ec.europa.eu/eurostat/databrowser/product/view/PROJ_19NANMIG?lang=en
